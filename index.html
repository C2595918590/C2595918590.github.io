Vue环境搭建教程：
  1. 下载Node.js
  2. $npm install -g vue-cli  || npm -v
  3. $npm install -g cnpm --registry=https://registry.npm.taobao.org
  4. $cnpm install -g vue-cli  || vue
  5. win+x 打开 || $set-ExecutionPolicy RemoteSigned
  6. $vue init webpack vue-demo
  7. $cnpm run dev
